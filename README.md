## About The Project

This is a responsive landing page for **"Tech Book Club"**, a subscription-based community platform for developers to read and discuss technical literature. The project focuses on a modern, professional UI with a **mobile-first** approach, ensuring the layout transitions smoothly from mobile devices to tablets and large desktop screens.

It demonstrates the use of semantic HTML5, advanced CSS layout techniques (CSS Grid & Flexbox), and SCSS nesting syntax for maintainable styling.

### Key Features

- **Mobile-First Workflow:** Styles are written for mobile first, then expanded for larger screens using min-width media queries (`768px` and `1440px`).
- **Semantic HTML:** Proper use of landmarks like `<header>`, `<main>`, `<section>`, and `<footer>` for accessibility.
- **Responsive Images:** Extensive usage of the `<picture>` tag to serve optimized image sizes and crops for different devices (Art Direction).
- **Advanced CSS:** Implementation of CSS Grid for the pricing cards and hero sections, and usage of background patterns/gradients for visual depth.
- **Custom Properties:** Usage of CSS variables for consistent color theming, typography, and spacing.

## Built With

- **[HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)** - Semantic structure and `<picture>` element usage.
- **[SCSS / SASS](https://sass-lang.com/)** - Utilized nesting syntax, mixins, and variables for cleaner stylesheets.
- **[CSS Grid](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout)** - Used for the main layout structure and membership options.
- **[Flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout)** - Used for component alignment, navigation, and feedback sections.
- **[Google Fonts](https://fonts.google.com/)** - Integration of "Martian Mono" and "Inter" for typography.
